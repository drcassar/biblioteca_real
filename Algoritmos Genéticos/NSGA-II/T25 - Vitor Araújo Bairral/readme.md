# Caminho_Aperfeicoador
Repositório virtual para a tarefa Caminho do Otimizador da disciplina de Redes Neurais e Algoritmos Genéticos da Ilum Escola de Ciência e Tecnologia. O objetivo desta tarefa é desenvolver um algoritmo genético para resolver um problema de interesse científico. Neste caso, foi implementado um algoritmo NSGA-II em python puro, bem como foi criada uma biblioteca de otimização genética multialgoritmo também em python puro a fim de otimizar parâmetros de construção de uma cavidade de radiofrequência utilizada em aceleradores de partículas.

## Estrutura do projeto:
caminho_aperfeicoador.ipynb: é o arquivo principal do projeto, onde estão todas as explicações e informações, bem como execuções de código. 

evolution.py e nsgaii.py: contêm códigos necessários para o funcionamento do projeto. São os arquivos que guardam os algoritmos.
